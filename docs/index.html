Local AI Box runs on your PC — upload PDFs and office files, index them locally, and chat with source citations. No cloud required for the default demo.

Download for Windows Launcher only (.exe)

Requires Docker Desktop (running). First start downloads images (~5–15 min).

Quick start

  1. 1

    Install Docker Desktop

    Windows install guide — WSL 2, engine running.

  2. 2

    Download & unzip

    Latest privateai-box-*-windows.zip from Releases.

  3. 3

    Run the launcher

    Double-click PrivateAIBox.exe (same folder as docker-compose.yml).

  4. 4

    Sign in & try it

    admin@example.com / changeme → Documents → Chat.

Requirements

  • Windows 10/11 (Mac/Linux: ./scripts/start.sh)
  • Docker Desktop + Compose v2
  • ~8 GB RAM · ports 3000 & 8000 free
  • Internet on first run

Image pull failed?

GitHub packages may be private until visibility is Public ( Packages). The launcher falls back to a slower local build if pull fails.
