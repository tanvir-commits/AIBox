Local AI Box runs on your PC — upload PDFs and office files, index them locally, and chat with source citations. Requires Docker (install once per machine).

Choose your platform:

Windows launcher only (.exe) All releases

First start downloads Docker images (~10–20 min). You need Docker installed and running before launching.

Windows — quick start

Download privateai-box-*-windows.zip from Releases, then double-click PrivateAIBox.exe.

  1. 1

    Install Docker Desktop

    Windows install guide — start it and wait until the engine is running.

  2. 2

    Download & unzip

    Latest release → privateai-box-*-windows.zip

  3. 3

    Run

    Double-click PrivateAIBox.exe in the unzipped folder.

  4. 4

    Sign in

    Browser opens http://localhost:3000 — admin@example.com / changeme → Documents → Chat.

Ubuntu — quick start

Download privateai-box-*-ubuntu.tar.gz from Releases (on the latest tagged release), or clone the repo. Then double-click localaibox or use the app menu shortcut.

  1. 1

    Install Docker (one-time)

    Extract the tarball, open a terminal in that folder, run: ./scripts/install-docker-ubuntu.sh then newgrp docker (or log out and back in).

  2. 2

    Download the app

    Releases → privateai-box-*-ubuntu.tar.gz
    Not on Releases yet? Use git clone https://github.com/tanvir-commits/AIBox.git and ./scripts/build-launcher.sh to create localaibox.

  3. 3

    App menu shortcut (optional, one-time)

    ./scripts/install-desktop-entry.sh → find Local AI Box in the Ubuntu app grid.

  4. 4

    Run

    Double-click localaibox in Files (allow “Run” / mark executable if asked), or launch from the app menu. Browser: http://localhost:3000

  5. 5

    Sign in & stop

    admin@example.com / changeme → Documents → Chat.
    Stop: ./localaibox -stop or app menu → Local AI Box (Stop).

Requirements

  • Windows 10/11 — Docker Desktop + Compose v2
  • Ubuntu 22.04 / 24.04 — Docker Engine + Compose v2
  • ~8 GB RAM · ~10 GB disk · ports 3000 & 8000 free
  • Internet on first run (image download)

Image pull failed?

GitHub packages may be private until visibility is Public ( Packages). The launcher falls back to a slower local build if pull fails.
